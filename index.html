Projects & Development

Explore the ongoing development of CRCON Player Info Admin tools designed to enhance your gaming experience. Our projects aim to provide comprehensive admin tools for server management and player information retrieval.

Player Info Admin Webapp

All in one tool to pull information from CRCON about players on the server. Features include synced login with Django auth and 3 tabs for reviewing players: TKs, Ban List, and VIP list.

HLL Admin Discord Bot

A discord slash commands bot to message in-game players. Features include chatting with in-game players using a discord channel and slash commands to warn players with default messages.
