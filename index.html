Projects & Development

Player Info Admin Webapp

  • All in one tool to pull information from CRCON about players on the server.
  • Synced login with Django auth.
  • 3 tabs for reviewing players:
    • TKs (team kills): players in active game which have more than 1 TK
      • Players ranked from most to least TKs.
      • Clicking on a player will load their chat history for the current session.
      • Ability to message player with default messages/warnings.
      • Ability to Flag player for review.
    • Ban List: List of all players on Server with more than 1 Ban
      • Rank players from most bans to least bans.
      • Ability to review past Bans with reason and comments.
      • Ability to Perma ban player.
      • Ability to Reset bans on player.
      • Ability to Flag player.
    • VIP list of all Players: List out all players that have VIP
      • Players can be sorted by soonest expiration to longest.
      • Ability to add players to VIP.
      • Ability to extend VIP time.
      • Ability to setup API call to check discord VIP (Role) status and create Flag for expired Discord VIP status.

HLL Admin Discord Bot

  • Chat with in-game players using discord channel.
  • Slash commands to Warn players with default messages.
